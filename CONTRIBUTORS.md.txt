# 课程学习目标和开源贡献计划

## 小组学习目标

- 掌握 Git 和 GitHub 的基本使用方法
- 学习开源软件开发流程
- 提高团队协作能力
- 完成本学期的开源项目贡献

## 开源贡献计划

### 第一阶段：基础知识学习
- 学习 Git 基本命令和工作流
- 熟悉 GitHub 平台的各项功能
- 了解开源社区文化和规范

### 第二阶段：实践参与
- 寻找适合的开源项目参与
- 从修复小 bug 或改进文档开始
- 学习代码审查流程

### 第三阶段：主导项目
- 发起自己的开源项目
- 或在现有项目中担任核心贡献者角色
- 指导新人参与开源社区

## 预期成果

- 每位成员至少向 3 个不同的开源项目提交 Pull Request
- 团队合作完成一个具有一定实用价值的小型开源项目
- 积累真实的开源社区协作经验
